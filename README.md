# SwiftTrip

A comprehensive travel application built with Flutter (Frontend) and Django REST Framework (Backend).

## Project Structure

- `backend/`: Django REST API with JWT authentication and travel management.
- `frontend/`: Flutter mobile/web application with profile management and travel search.

## Getting Started

### Backend
1. Go to the `backend/` directory.
2. Activate your virtual environment: `venv\Scripts\activate`.
3. Run the server: `python manage.py runserver 0.0.0.0:8000`.
- To receive OTPs, create a free account at Mailtrap.io, copy your Inbox credentials into a new .env file, and the app will send verification codes to your Mailtrap dashboard

### Frontend
1. Go to the `frontend/` directory.
2. Run the app: `flutter run -d chrome`.
